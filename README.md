Um jogo de plataforma onde vc controla um Pirata Esqueleto com o objetivo de pegar as moedas de ouro para ganhar, enquanto evita encostar nos slimes.

Para jogar vc deve entrar no arquivo "ExePiratexOficial.rar" e clicar em "download raw file" ou instalar todos os aquivos em .zip (isso fará o download dos assets, packages e projects settings).
Depois de instalar extraia o arquivo .rar e entre na Pasta criada "ExePiratexOficial" e entre no executável "Projeto_segunda".
Caso entre na tela de "Vitória!" ou "Derrota!" feche o jogo e abra novamente para jogar.
